# NoSQL

## Description
This project is part of the Back-end curriculum. It covers the fundamental concepts of NoSQL databases using **MongoDB**.

## Requirements
* **OS:** Ubuntu 18.04 LTS
* **Database:** MongoDB 3.6.3
* All files should end with a new line.

## Project Structure
* `0-list_databases`: List all databases.
* `1-use_or_create_database`: Create/use a database.
* `2-insert`: Insert a document.
* `3-all`: List all documents.
* `12-log_stats.py`: Python script for log stats.
